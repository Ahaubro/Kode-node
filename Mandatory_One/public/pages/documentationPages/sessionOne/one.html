Rest API


There is some conventions when it comes to Rest Api. When you make an endpoint folow these rules: And remember - Ressource names should be nouns in plural.

  • Get all - GET without ID
  • Find by id - GET with ID
  • Create new - POST without ID
  • Update ressource - PUT with ID
  • Update ressource partially- PATCH with ID
