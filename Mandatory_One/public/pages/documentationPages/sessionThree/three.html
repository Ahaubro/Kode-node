Get requests

When sending data to our server with a GET method, we need to use a pathvariable or a query String to send data.

  • Path variable - We can use a path variable in our endpoint, by adding it to the path after -> "app.get('/beers/:id', (req, res) =>"
  • Query String - We can use a query String, by using key & value in a url. When our path ends, we can write the query String ?key=value&key2=value2
