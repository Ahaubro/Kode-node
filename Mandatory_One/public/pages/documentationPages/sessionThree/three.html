Nodemon

Nodemon is a gift to developers. We can use nodemon to continuously build our project, so we dont have to rerun our project, to use the changes we are making automatically. The nodemon tool helps develop node.js applications by automatically restarting the application when changes are detected in the files. To use nodemon we follow these steps:


  • npm install -g nodemon
  • npx nodemon
