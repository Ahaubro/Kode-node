Terminal commands



Here is a list of some of the terminal commands we use in Node.JS

  • node filename - Runs file
  • npm install modulename - Installs a module
  • npm install - Installs all dependencies
  • npm init - Initializes a node project
  • nodemon.cmd filename - Runs file with nodemon
  • npm run scriptname - Runs a script from the package
  • git push - Push to GitHub
  • git clone urlname - Clone project from remote repository
  • git add filename - Add file for next commit
  • git commit - To commit changes
